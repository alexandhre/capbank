# CapBank - CapBank

## Quick start

```bash
# clonar o repositório
git clone https://github.com/alexandhre/pokefan.git

# mudar de pasta
cd capbank/app-capbank

# instalar dependências
npm install

# startar projeto
npm run dev

### Requisitos
O que você precisa para executar este aplicativo:
* VueJs CLI 3

### Descrição
* Solução desenvolvida em VueJs atendendo as seguintes funcionalidades:
- Consultar Saldo e Extrato;
- Saque;
- Depósito;

Arquitetura baseada em componentes utilizando as diretivas e padrões aplicados ao VueJs.
